# QQbot_pansou

基于AstrBot的网盘资源搜索机器人插件，支持在QQ群内搜索各类影视资源并获取下载链接。

## 功能特性

- 🎬 **资源搜索**：通过关键词搜索影视资源
- 📦 **链接获取**：自动提取有效下载链接
- 📱 **智能交互**：支持群成员加入欢迎、被@时发送使用说明
- 📄 **结果分页**：支持多页搜索结果的浏览
- ❌ **错误提示**：搜索失败时提供详细的使用指导

## 安装方法

1. 确保已安装AstrBot机器人框架
2. 将本插件目录放置于AstrBot的插件目录中
3. 重启AstrBot，插件将自动加载

## 使用说明

### 1. 资源搜索

**基础格式**：`搜+剧名`

**示例**：`搜仙逆`

**自定义网盘搜索格式**：
- 百度网盘：`百度+剧名`，例如：`百度仙逆`
- 夸克网盘：`夸克+剧名`，例如：`夸克仙逆`
- UC网盘：`UC+剧名`，例如：`UC仙逆`
- 迅雷网盘：`迅雷+剧名`，例如：`迅雷仙逆`

系统将自动搜索相关资源并返回结果列表，例如：
```
🔍 搜索结果（共 5 个，第 1/1 页）

【1】仙逆 - 百度网盘
    📦 百度云
------------------------

【2】仙逆 - 阿里云盘
    📦 阿里云
------------------------

【3】仙逆 - 夸克网盘
    📦 夸克云

💡 输入「下一页」或「上一页」翻页
💡 输入「第X个」或「X」选择资源（如：第1个、1）
```

### 2. 选择资源

在搜索结果出来后，输入数字或"第X个"选择对应资源，例如：

**示例**：`2` 或 `第2个`

系统将返回该资源的详细下载链接。

### 3. 分页浏览

当搜索结果较多时，系统会自动分页显示。可以使用以下命令浏览不同页面：

- 查看下一页：`下一页`
- 查看上一页：`上一页`

### 4. 被@时的帮助

在群内@机器人时，机器人会发送使用说明：

```
想要看啥剧，输入搜+剧名发群里并输入数字即可获取链接
如 "搜仙逆" 跳出来的对话 如 "2"
PS:搜索功能是机器人回复的，群主没法实时看群，有问题@群主等群主来解决就行
```

### 5. 新成员加入欢迎

当有新成员加入群聊时，机器人会自动发送欢迎消息：

```
@新成员 欢迎小伙伴，想要看啥剧，输入搜+剧名发群里并输入数字即可获取链接

PS:搜索功能是机器人回复的，群主没法实时看群，有问题@群主等待处理
```

## 搜索技巧

为了提高搜索成功率，请遵循以下搜索技巧：

- ✅ 剧名宁少写，不多写、错写
- ✅ 不要使用标点符号
- ✅ 不要包含演员名
- ✅ 不要包含第几季、第几集等信息

**示例**：
- 正确：`搜仙逆`
- 错误：`搜仙逆电视剧`、`搜仙逆第一季`、`搜仙逆（主演：XXX）`

## 搜索失败提示

当搜索失败时，系统会显示以下提示：

```
>>>>查询失败<<<<
--------------------
剧名宁少写，不多写、错写
不要标点、演员名、第几季
如再查询不到@群主帮你找
```

## 注意事项

1. 搜索功能由机器人自动回复，群主无法实时查看所有搜索内容
2. 若遇到搜索问题，请@群主等待处理
3. 请遵守相关法律法规，合理使用本插件
4. 请勿滥用搜索功能，避免给服务器造成过大压力

## 常见问题

**Q: 为什么搜索不到想要的资源？**
A: 请尝试简化剧名，不要包含标点、演员名或季节信息，如仍无法找到可@群主协助。

**Q: 搜索结果中的链接无法打开怎么办？**
A: 网盘链接可能存在时效性问题，建议尝试搜索其他资源或@群主反馈。

**Q: 如何获取更多资源？**
A: 可以尝试使用不同的关键词组合，或@群主请求添加特定资源。
